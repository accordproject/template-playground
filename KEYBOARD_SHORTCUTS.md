# Keyboard Shortcuts Guide

This document outlines all available keyboard shortcuts in the Accord Project Template Playground.

## Navigation & Panel Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Alt + E** | Toggle Editors Panel | Show/hide the code editor panel containing model, template, and data editors |
| **Alt + P** | Toggle Preview Panel | Show/hide the preview panel that displays the rendered agreement |
| **Alt + B** | Toggle Problems Panel | Show/hide the problems/errors panel for debugging and error messages |
| **Alt + A** | Toggle AI Chat Panel | Show/hide the AI assistant chat panel for interactive assistance |
| **Alt + O** | Toggle FullScreen Modal | Open/close the fullscreen modal showing the rendered agreement output |

## Appearance & Theme

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Alt + D** | Toggle Dark Mode | Switch between light and dark theme for better accessibility |

## Workspace Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Alt + S** | Copy Share Link | Generate and copy a shareable link to your clipboard (includes current workspace state) |
| **Alt + T** | Toggle Tour | Start or close the interactive tutorial |
| **Alt + L** | Navigate to Learn | Go to the learning path page at `/learn/intro` |

## External Links

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Alt + 1** | Open Accord Project GitHub | Visit the Accord Project GitHub repository in a new tab |
| **Alt + 2** | Open Accord Project Discord | Join the Accord Project Discord community in a new tab |

## How to Use

1. **Keyboard Shortcuts**: Press the Alt key along with the specified character or number to trigger an action
2. **Browser Compatibility**: These shortcuts work in most modern browsers
3. **Conflicts**: If a shortcut conflicts with your browser or OS shortcuts, the browser/OS action may take precedence
4. **Confirmation**: Actions like "Reset Workspace" (Alt + K) will prompt you to confirm before proceeding

## Tips & Tricks

- Use **Alt + D** frequently to toggle dark mode for comfortable working in different lighting conditions
- Use **Alt + S** to quickly share your work with others - the link includes your complete workspace state
- Use **Alt + T** to revisit the tutorial if you need to refresh your memory on how to use a feature
- Use **Alt + E**, **Alt + P**, and **Alt + B** to focus on different aspects of your work
- Use **Alt + A** to quickly access AI assistance without navigating through menus
